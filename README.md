# FileOrganizer
A CLI tool written in Python that automatically sorts files into subfolders by type.

## Usage
```bash
python file-org.py
```

## Supported File Types
| Category | Extensions |
|----------|------------|
| Images | .jpg .jpeg .png .gif |
| Videos | .mp4 .avi .mkv .mov |
| Documents | .pdf .docx .txt .md .pptx |
| Audio | .mp3 .wav |
| Code (C++) | .cpp |
| Code (Python) | .py |
| Web | .html .css .js |
| Compressed | .zip .rar .tar .gz .7z |
| OS Images | .iso |
| Other | anything else |
